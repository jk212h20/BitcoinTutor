2025

  Analysis of syncing full nodes without downloading old witnesses


2021

  Bitcoin Core #21009 removes logic needed to upgrade a pre-segwit node to segwit


2020

  BIPs #947 updates the BIP325 specification of signet to allow segwit-style virtual transactions
  Bitcoin Core #18044 adds support for announcing and requesting transactions by their segwit wtxid
  BIPs #933 adds BIP339 for transaction relay announcement using segwit wtxids
  Fee overpayment attack on multi-input segwit transactions


2019

  Question: Are there still miners or mining pools which refuse to implement SegWit?
  Bitcoin Core #15846 relays and mines transactions paying any segwit address version
  Bitcoin Core #14039 causes Bitcoin Core to reject legacy transactions encoded as segwit
  Bitcoin Core #14811 updates the getblocktemplate RPC to require that the segwit flag be passed


2018

  Temporary reduction in segwit block production
  Bitcoin Core #14454 adds support to the importmulti RPC for segwit addresses and scripts
