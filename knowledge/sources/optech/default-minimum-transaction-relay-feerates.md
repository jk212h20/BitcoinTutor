/ home / topics / 


    Default minimum transaction relay feerates
    
  

  
    




Default minimum transaction relay feerates are the policy implemented by nodes for ignoring individual unconfirmed transactions whose feerate is below a certain amount.  For Bitcoin Core, this threshold has for several years been 1 sat/vbyte.
