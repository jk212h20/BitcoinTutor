Also covering symbll, bllsh, and BTC Lisp

Basic Bitcoin Lisp language (bll) is a proposed scripting language that could be added to Bitcoin in a soft fork.  Formerly called BTC Lisp and conceptually based on Chia Lisp, it’s part of a set of tools that includes symbll (a miniscript-like compiler of higher-level Lisp to lower-level bll) and bllsh (a REPL for trying and debugging symbll and bll).
